<h1>Real Time Laravel Chat group using laravel websocket (live chat system) </h1>

<h4>Getting start</h4>
<ul>
    <li>Composer install</li>
    <li>npm install</li>
    <li>php artisan migrate</li>
    <li>npm run dev</li>
    <li>php artisan serve</li>
    <li>php artisan websockets:run</li>
</ul>
